# PipoApp - Frontend
---
Este es el frontend de PipoApp, una aplicación para el registro de clientes con crédito en una tienda de ropa. El frontend está desarrollado utilizando tecnologías modernas como React y TailwindCSS.

## Características

	•	Login de usuarios administrativos: Permite a los administradores iniciar sesión con su nombre de usuario y contraseña para acceder al sistema de gestión de clientes.
	•	Registro y gestión de clientes: El administrador puede agregar, editar, eliminar y ver clientes con crédito en la tienda.
	•	Visualización de productos a crédito: Permite al administrador gestionar los productos que los clientes han adquirido a crédito.
	•	Historial de crédito: Cada cliente tiene un historial de movimientos de crédito.
	•	Interfaz de usuario moderna: El diseño utiliza TailwindCSS para crear una interfaz limpia y moderna.

## Tecnologías Usadas

	•	React: Biblioteca de JavaScript para construir interfaces de usuario.
	•	TailwindCSS: Framework CSS para diseño de interfaz de usuario.
	•	React Router: Librería para manejar la navegación entre las páginas del frontend.
	•	Axios: Cliente HTTP para hacer peticiones a la API backend.
